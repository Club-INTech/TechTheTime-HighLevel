Detailed Description

A namespace scom is a shortcut for serial communication.

It introduces a scom::BlockingModes, scom::SerialPort and other members, that are required for using TechTheTime-Shared and Key-To-Order

This libraries allow to communicate with microcontroller. scom::SerialPort introduces some serial communication basics (using termios) and some specific methods used as communication basis of Key-To-Order
