\mbox{\hyperlink{manager_8cpp}{manager.\+cpp}} is an entry point of the manager node. 

We initialize ros2 client library \href{https://docs.ros2.org/foxy/api/rclcpp/index.html}{\texttt{ rclcpp}} Then we create a shared pointers to clients which inherit from \mbox{\hyperlink{classClientT}{ClientT}}. We call \mbox{\hyperlink{classClientT_a5652e544df0a717cf97a6bc5c5df04c5}{Client\+T\+::set\+\_\+shared}} to set a self shared pointer which is needed for ros2 client methods. Finally we can use \mbox{\hyperlink{classClientT_aed77067671c316fe4e750278193dc90c}{Client\+T\+::wait\+\_\+for\+\_\+connection}} method to connect to the service. We can then use \mbox{\hyperlink{classClientT_a722f4ae9c93fd8ad61756155bb1e79be}{Client\+T\+::send}} method.
