Detailed Description

template<typename Treq, typename Tres>
struct OrderBinder< Treq, Tres >

OrderBinder is used by ActionService in order to execute callback attached to an order code.

You can bind any of OrderCodes(order_codes.hpp) to any function with the OrderBinder::order_functor_T signature and then execute it by providing an order_code and parameters.

Template Parameters
TreqThe type of request for the order
TresThe type of response of an order
